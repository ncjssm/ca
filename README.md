# XP Chat MVP

Fullstack MVP with Express + SQLite + Socket.io and a React + Vite XP-inspired UI.

## Quick start

Server:

```bash
cd server
npm install
npm run dev
```

Client:

```bash
cd client
npm install
npm run dev
```

Open `http://localhost:5173`.

## Notes

- Database file: `server/db/chat.sqlite` (stored via `sql.js` for no native builds)
- Sessions are in-memory (dev only); set `SESSION_SECRET` in production.
- Avatar can be a URL or a `data:image/...` data URI.
